# Fake News Detection Model
A supervised machine learning model that classifies news articles as real or fake using  Logistic Regression and Natural Language Processing (NLP).

## Results
Accuracy  | 98.75% 
Precision | 99.08% 
Recall    | 98.53% 

## Built With
- Python
- scikit-learn
- pandas & numpy
- TF-IDF Vectorisation
- Matplotlib & Seaborn

## Dataset
[Fake News Detection Dataset](https://www.kaggle.com/datasets/emineyetm/fake-news-detection-datasets/data) 
by emineyetm — 40,000+ labelled news articles
